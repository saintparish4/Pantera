# PANTERA - Dynamic Pricing Engine API

> "Stripe for Pricing" - Developer-focused API for dynamic pricing calculations

## Overview

PANTERA is a powerful, developer-friendly API that enables dynamic pricing calculations for your applications. Built with Go and PostgreSQL, it provides flexible pricing strategies and real-time price optimization.

## Quick Start

### Prerequisites

- **Go** 1.24.4+
- **PostgreSQL** 14+
